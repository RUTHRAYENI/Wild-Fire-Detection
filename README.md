# 🌿 Turmeric Leaf Disease Detection using EfficientNet & ResNet

## 📌 Project Overview
This project presents an intelligent system for identifying turmeric leaf diseases using deep learning techniques. By leveraging advanced convolutional neural network architectures such as EfficientNet and ResNet, the system aims to assist farmers and agricultural experts in early and accurate disease detection, thereby improving crop productivity and reducing losses.

---

## 🎯 Objectives
- To classify turmeric leaf diseases using image-based deep learning models  
- To compare the performance of EfficientNet and ResNet architectures  
- To provide a reliable and scalable solution for agricultural disease diagnosis  

---

## 🧠 Models Used
- EfficientNet – Optimized for high accuracy with fewer parameters  
- ResNet (Residual Network) – Helps in training deeper networks effectively  

---

## 📊 Dataset
- Image dataset of turmeric leaves  
- Contains multiple disease categories (approximately 15 classes)  
- Includes both healthy and diseased leaf samples  

> 📁 Note: Dataset is not included in this repository due to size limitations.
