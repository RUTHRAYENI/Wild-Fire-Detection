# 🔥 Wildfire Detection System using Deep Learning

## 📌 Project Overview
This project presents an automated wildfire detection system using deep learning techniques. The system analyzes images to detect the presence of fire or smoke at an early stage, helping to prevent large-scale environmental damage and improve disaster response time.

---

## 🎯 Objectives
- To detect wildfire or smoke from image data  
- To build an efficient and accurate deep learning model  
- To provide early warning support for disaster management systems  

---

## 🧠 Models Used
- EfficientNet – High accuracy with optimized performance  
- ResNet (Residual Network) – Enables deep learning with skip connections  

---

## 📊 Dataset
- Image dataset containing wildfire and non-wildfire scenes  
- Includes images of fire, smoke, and normal environments  
- Used for training and testing the model  

> 📁 Note: Dataset is not included in this repository due to size limitations.
